第一章 文件处理命令
ls 
参数
-a 所有文件,包含隐藏文件 .开头表示隐藏文件
-l 详细信息  可以指定目录比如 ls-l /etc/
-d 查看目录本身属性而不是目录里面的文件
-h  h表示人性化显示 大部分命令都可以用
-i 	显示inode,inode表示文件用的

--目录处理命令--
mkdir -p [目录名]
-p 递归创建
cd - 进入上次进入的目录
cd \home\  绝对路径
-常用一级目录-
/boot 存放内核以及启动所需的文件等
/dev 存放设备文件
/etc 存放系统的配置文件
/home 用户文件的主目录，用户数据存放在其主目录中
/lib 存放必要的运行库
/mnt 存放临时的映射文件系统，我们常把软驱和光驱挂装在这里的floppy和cdrom子目录下。
/proc 存放存储进程和系统信息
/root 超级用户的主目录
/sbin 存放系统管理程序
/tmp 存放临时文件的目录
/usr 包含了一般不需要修改的应用程序，命令程序文件、程序库、手册和其它文档。
/var 包含系统产生的经常变化的文件
--链接命令--
ln -s [原文件] [目标文件] 
功能描述：（硬链接相当于新提供了个入口访问文件,删除不影响原有文件,不能跨分区创建,不能针对目录只能文件）--硬链接不推荐
						软链接相当于windows快捷方式
选项 -s 创建软链接（不加 -s 就是硬链接）

第二章 文件搜索命令
--文件搜索命令 locate 
locate [文件名]
在后台数据库中搜索,效率更高,新建文件不能自动更新数据库(一天一更新,updatedb强制更新)
根据/etc/updatedb.conf配置搜索策略

--命令搜索命令whereis与which
whereis ls
-b:只查找可执行文件
-m:只查找帮助文件
which ls
whatis ls

--find 【搜索范围】 【搜索条件】
例如：find /root -name "ab[cd]"
*  								任意内容
？ 								任意一个单独字符
[] 								匹配任意一个中括号内的字符
-inname 					不区分大小写
-user   					按所有者搜索
-nouser 					没有所有者的文件（常用）
-mtime  					修改时间  单位天 +10--》10天前  -10--》10天内   10--》10天当天
-ctime  					改变文件属性
-atime  					文件访问时间
-size   					文件大小 
-inum   					i节点
-a                相当于and  -size +20k -a -size -40k(大于20小于40k)
-o                相当于or
-exec 命令 {}\;   对搜索结果执行操作 -exec ls -lh {}\;(结果集执行ls -lh)

--grep [选项] 字符串 文件名
#在文件当中匹配符合条件的字符串
选项：
-i 忽略大小写
-v 排除制定字符串
